# Diabetes Prediction using Machine Learning

A Machine Learning project that predicts whether a person is likely to have diabetes based on medical attributes. The project includes data preprocessing, model training, evaluation, and prediction using Python and Scikit-learn.

---

## Project Overview

Diabetes is one of the most common chronic diseases worldwide. Early prediction can help in timely diagnosis and treatment.

This project uses Machine Learning classification algorithms to predict diabetes based on patient health information.

---

- Data preprocessing

## Dataset

The project uses the **Pima Indians Diabetes Dataset**.

Features include:

- Pregnancies
- Glucose
- Blood Pressure
- Skin Thickness
- Insulin
- BMI
- Diabetes Pedigree Function
- Age

Target:

- Outcome
  - 0 = Non-Diabetic
  - 1 = Diabetic

## Installation

Clone the repository

```bash
git clone https://github.com/husnamea-it/diabetes-project.git
```

Move into the project folder

```bash
cd diabetes-project
```

Install required packages

```bash
pip install -r requirements.txt
```

---

## Run the Project

Open the Jupyter Notebook

```bash
jupyter notebook
```

or run the Python file

```bash
python diabetes_prediction.py
```
